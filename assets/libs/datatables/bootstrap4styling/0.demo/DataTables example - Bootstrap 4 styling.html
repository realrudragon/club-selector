Bootstrap 4 styling

This example shows DataTables and the Responsive extension being used with Bootstrap 4 providing the styling. The DataTables / Bootstrap integration provides seamless integration for DataTables to be used in a Bootstrap 4 page.

Note that the dt-responsive class is used to indicate to the extension that it should be enabled on this page, as responsive has special meaning in Bootstrap. The responsive option could be used as an alternative if you prefer to use Javascript initialisation options.
